# WeixinAutoChat(基于java8)
微信机器人

```java
MessageHandler handler = new MessageHandler();
WeixinAutoChat chat = new WeixinAutoChat(handler);
handler.chat = chat;
chat.login();
```

----------------

常见的场景

* 定时发消息给你的朋友
* 自动聊天发文本，表情，图片
* 自动拉群
* 逗人玩
* ...
